      Custom Mode Config

      ▾
      Each row in the spreadsheet is one scenario.
      The first column is Scenario (scenario number/label).
      Each subsequent column header is the Name you defined for each input variable (e.g. K_droop).
      The sheet name is always Sheet1 (the template generates exactly one sheet).
      Click Download Template to get a pre-filled template based on your input variable names.

      Output Variables

      ▾
      Attribute: read a scalar result attribute from a PF element after solving (obj.GetAttribute).
      Timeseries: extract a scalar metric from the dynamic result file (ElmRes). Requires a dynamic study type.
      Custom Calculation: user-defined Python function that can reference other input/output variable values.
      Variable names must be unique across all input and output variables.
