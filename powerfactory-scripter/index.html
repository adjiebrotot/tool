    Custom Mode Config

    ▾
    Each row in the spreadsheet is one scenario.
    Each column header must be Object.Variable (e.g. WFController.ElmDsl.K_droop).
    Click Download Template to get a pre-filled header row based on your input variables.

    Custom Calculation: user-defined Python function referencing other outputs.
