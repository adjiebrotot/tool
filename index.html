Maintenance notes

This page is built so search engines can read the tool content directly from the HTML, while the filters are handled with lightweight JavaScript.

  • To add a new tool, duplicate a card and update its title, subtitle, status, description, tags, and link.
  • Supported statuses are Active, Coming soon, and Deprecated.
  • Tags are read automatically from each card, so new tags will appear in the filter without extra setup.
  • For best SEO, also update the JSON-LD block in the <head> when adding major new tools.
